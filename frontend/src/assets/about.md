## Welcome to Eden's Garden
Eden's Garden is a small-scale personal self-hosting project, run for the purpose of connecting a small community of friends, and developing my own skills in web development, system administration, and data analysis.
Eden's Garden currently hosts a variety of software, including:
- Cryptpad
- Mattermost
- Mirotalk
- Radicale

Eden's garden uses a tech stack based around
- Containerization via Docker
- Reverse proxying via Caddy
- Observability stack via Grafana

and is powered by Rocky Linux on an OVHCloud VPS in Quebec.
